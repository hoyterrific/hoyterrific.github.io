About: An app developed in Java and XML for android. Targeted for users that have trouble
waking up in the morning. When the alarm goes off, the user must complete a series of
games to turn off the alarm.

Languages: Java and XML

Team: Alex Hoyt, Olivia Thom and Raquel Robinson

Roles: Developed an alarm app from scratch using android studio. Implemented a notification
system, simple puzzle game, app layout, button mechanics and sound.
